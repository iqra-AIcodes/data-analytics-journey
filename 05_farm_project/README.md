# 05 — Farm Analytics Project 🌾

Real-world exploratory data analysis using operational data 
from a family-owned agricultural and livestock business.

## Why This Project
Most data portfolios use the same Titanic or Iris datasets. 
This project uses real operational logs from a working farm — 
making it unique and genuinely useful.

## Questions Being Investigated
- How does livestock output change across seasons?
- What is the feed-to-output efficiency ratio?
- Are there anomalies or outlier events in the records?
- What patterns exist in operational costs vs output?

## Tech Stack
- Python · pandas · NumPy
- SQL · SQLite
- Matplotlib · Seaborn · Plotly

## Project Structure
| Folder/File | Description |
|-------------|-------------|
| data/ | Anonymized sample CSV files + data dictionary |
| notebooks/ | Analysis notebooks in sequence |
| reports/ | Summary of findings |
