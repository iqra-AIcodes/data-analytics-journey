# 04 — Data Visualization

Building clear, informative charts and dashboards.

## Topics Covered
- matplotlib: line, bar, scatter, histogram
- seaborn: heatmaps, pairplots, distribution plots
- Plotly: interactive charts
- Dashboard basics with Streamlit

## Files
| File | Description |
|------|-------------|
| matplotlib_basics.ipynb | Core chart types |
| seaborn_plots.ipynb | Statistical visualizations |
| plotly_interactive.ipynb | Interactive charts |

## Design Principles
- Every chart has a title, axis labels, and units
- Color used to highlight insight, not decorate
- Simple is better than complex
